## Installation

As described
[here]({{ "/docs/analysis-flow" | relative_url }})
you may need to run Stage 1 Pipelines and/or Stage 2 Apps, which 
have distinct installation needs. 

How you install will also depend on whether you have
Singularity/Apptainer container support available on your HPC server 
and whether the tool suites you use offer containerized code.
